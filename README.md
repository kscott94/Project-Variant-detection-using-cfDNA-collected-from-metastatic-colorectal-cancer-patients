# GTM-Project

### Project Description

I am reproducing part of the variant analysis from Lim et al. (2021), focusing on Patient 42. Using cell-free DNA (cfDNA) samples, I am examining chromosome 17 variantsin mCRC patients across three time points relative to anti-EGFR monoclonal antibody (cetuximab) treatment: before treatment, during treatment, and at disease progression.
